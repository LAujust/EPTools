Quick Start
=======================

This is a quick guidance for using the EPTools to aid your work as Transient Advocate (TA) and subsequently data analysis. 


Installation
-----------------------

- From Source

.. code-block:: bash

    git clone https://github.com/LAujust/EPTools.git

    cd EPTools/

    pip install .

- From pip (under assessment)

.. code-block:: bash

    pip install eptools
